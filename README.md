# Portscanner

Portscanner is python based port and services scanner
It can be used with a variety of options
Speed of scanning can also be controlled.

# Installation

sudo git clone https://github.com/Amishmishra20/PortScanner.git

cd PortScanner/

sudo chmod +x setup.sh

sudo bash setup.sh

# Mandatory Argument

-ip or --host - Target IP To Scan

# Optional Arguments

-h or --help - For Help

-t or --threads - threads number

-s or --start_port - Port Number to start scan From

-e or --end_port - Port Number To end scan At
